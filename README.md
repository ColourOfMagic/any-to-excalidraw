# Any to Excalidraw

A web application that converts text input into Excalidraw elements.

## Features

- Convert text lines to Excalidraw text elements
- Interactive Excalidraw canvas for editing
- Clean and simple interface

## Usage

1. Enter text in the sidebar input area
2. Each line will become a separate text node
3. Click "Convert" to generate Excalidraw elements
4. Edit and manipulate the elements in the canvas

## Tech Stack

- React + TypeScript
- Vite
- Excalidraw
- CSS Modules
